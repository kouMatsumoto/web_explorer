## Preparation

- Install Node.js and npm
- Install Prettier
  - npm i -g prettier

## Commands

- develop
  - firebase emulators:start
  - Run application by using IDE
- format
  - flutter format .
  - flutter analyze
- build
  - flutter build apk
  - connect a device to install and get device id from `flutter devices`
  - `flutter install -d ${DEVICE_ID}`
